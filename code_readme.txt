code/C1_initial_process_dewey_data_into_JEA.R: This code takes as input all the raw .parquet files from dewey and consolidates them into one file, keeping relevant variables, and subsetting the sample only to areas serviced by JEA. The output is dewey_jea_service_area.parquet (and .dta). The script was originally written by Emma DeAngeli and then last updated by Martin Navarrete on 6/16/2026

code/exploratory_data_analysis_dewey_jea.do: This .dofile uses as input, dewey_jea_service_area.dta. The goal of this script is to do a first exploration of the dataset and see summary statistics of key variables like home charasteristics, tax delinquency, among others. The output is all saved as a log file in output\logs\eda_dewey_jea.log. The author of this script is Martin Navarrete and it was last edited on 6/18/2026

code/jea_arrears_data_exploration.R: This script explores the JEA arrears aging report data. It loads the non-PII arrears dataset from data/intermediate/JEA/aging_report_nonpii.parquet, cleans and renames selected variables, converts arrears balance fields from character/string format into numeric values, and produces summary statistics by service type. The script was authored by Martin Navarrete and was last updated on 6/26/2026

code/jea_bill_reconstruction.R: This script reconstructs electricity bills for RES10 customers using electricity consumption from data/intermediate/JEA/elec_df_agg.parquet and the monthly JEA electricity rates from data/raw/rates/electricity_rates_JEA_2019_2026.xlsx. It reconstructs final bill including tax, and then compares the reconstructed bill to the observed BILL_AMOUNT using summary statistics for absolute reconstruction errors, percent reconstruction errors, reconstruction accuracy by tax area. The main output is the dataset elec_res10_reconstructed, which contains the reconstructed bill variables and reconstruction error measures. The script was authored by Martin Navarrete and last edited on 7/6/2026

code/jea_billing_data_elec_aggregation.R:  This script aggregates the JEA monthly electricity billing data from data/intermediate/JEA/monthly_electricity_bill_data_nonpii.parquet. It groups duplicate rows that share the same billing/account/service-point characteristics and the same BILL_AMOUNT, sums their CONSUMPTION and records how many rows were aggregated. The main output is /elec_df_agg.parquet. The script was authored by Martin Navarrete and was last edited on 6/7/2026.

code/jea_disconnections_data_exploration.R: This script explores the JEA residential electric non-payment disconnections data from data/raw/JEA/PRR-J016546-Household_Residential_Electric_Disconnects_Non-Pay.csv. It summarizes the number of observations, accounts, date range, and disconnections per account, and creates monthly and month-of-year summaries of disconnections. The script was authored by Martin Navarrete and was last edited on 7/6/2026.
